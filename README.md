
# CRUD em Laravel
## Projeto dividido em dois diretórios

### Back-end - PHP(Laravel)
Abrir o diretório do back-end pelo cmd e digitar os seguintes comandos:

 - composer install
 - php artisan migrate
 - php artisan serve

### Front-end UI(Vue)
Abrir o diretório do front-end pelo cmd e digitar os seguintes comandos:

 - npm i
 - npm run serve
